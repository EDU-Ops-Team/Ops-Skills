# Vendor Return Template

This is the template vendors fill out after their site visit. It is structurally identical to the AI pre-report but with two additions:

1. A `Source` column on every Section 1 table row (B/C confidence items arrive pre-labeled "AI found: X — please verify")
2. A `Verification Notes` field on every Section 2 task card

Vendors receive this as their working document. The AI pre-populates it via Mode 2 (Vendor Packet). This blank template is the reference for what a completed return should look like.

---

```
# Vendor Site Return
## [Address]
### Visit Date: [Date] | Vendor: [Name / Company]

---

## INSTRUCTIONS

Rows labeled **"AI found: X — please verify"** need your independent confirmation.
Fill in the Source column: where did you verify this? (e.g., "FL statute 633.202",
"staff call 4/3/26", "observed on site", "county assessor portal").

Leave Source blank only if you did not verify the item at all — we need to know the
difference between "confirmed" and "carried forward."

Complete every Section 2 task card. These are the field-only items the AI could not
determine remotely.
```
